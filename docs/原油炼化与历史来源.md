# 原油、炼化与历史模块来源

## 调研目标

第三阶段回答四个问题：

1. 为什么不同原油不能被视为同一种商品？
2. 为什么同一桶重油在不同炼厂具有不同经济价值？
3. 为什么 WTI、Brent、Dubai/Oman 和 Murban 不是可随意互换的“油价”？
4. 为什么历次石油危机不能只按油价曲线寻找相似年份？

## 原油与炼化

### EIA：原油并不相同

来源：<https://www.eia.gov/energyexplained/oil-and-petroleum-products/refining-crude-oil-inputs-and-outputs.php>

采用内容：API 度、含硫量、轻重甜酸，以及轻油更容易通过简单蒸馏形成高价值产品、重油需要额外转化装置的基本关系。

### EIA：炼化三步骤

来源：<https://www.eia.gov/energyexplained/oil-and-petroleum-products/refining-crude-oil-the-refining-process.php>

采用内容：分离、转化、处理三步；裂化使用热、压力、催化剂和氢气把重分子转为更轻产品。

### EIA：代表油种品质坐标

来源：<https://www.eia.gov/todayinenergy/detail.php?id=7110>

采用内容：密度与含硫量是最重要的两个公开品质坐标之一；轻质低硫油通常更容易加工，但价格还会受地点与市场结构影响。

## 模型边界

### 原油典型值

同名原油的 API 度和含硫量会随货批、装港、混兑和测量方法发生变化。页面数值用于比较典型位置，不是验货标准。

### 炼厂产品结构

产品条是方向性教学模型。它没有替代原油评价、物料平衡、炼厂线性规划模型或上市公司的经营披露。

### 历史冲击指纹

1—5分仅用于比较供给、需求、物流和政策的相对重要性，不代表对价格涨跌的统计贡献。
