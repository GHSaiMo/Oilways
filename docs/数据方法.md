# 油脉数据方法

## 信息等级

页面信息分为三类，并在展示时保持边界清楚：

1. **事实**：来源机构直接公布的流量、消费、产量和价格数据。
2. **估算**：基于公开事实计算出的比例、区间或情景结果。
3. **教学模型**：用于说明机制的简化公式，不作为油价或收益预测。

网站使用对应的可见标签：

- **结构性事实**：外部原始来源可核验，并标注年份与口径。
- **研究框架**：由多份研究材料归纳出的分析顺序，不是统计结论。
- **教学模型**：输入与公式透明，输出用于理解敏感度，不是预测。

## 来源优先级

1. EIA、IEA、IMF、OPEC、各国统计与监管机构
2. Energy Institute 等长期统计项目
3. 上市公司年报、业绩材料和交易所文件
4. 学术论文、央行及研究机构
5. 新闻和二手数据库仅作为线索，不作为关键数字的唯一来源

## 当前核心口径

| 指标 | 原型数值 | 年份 | 用途 |
| --- | ---: | --- | --- |
| 霍尔木兹石油流量 | 20.9 mb/d | 2023 | 首屏与压力测试基线 |
| 霍尔木兹全球占比 | 约 20% | 2023 | 解释咽喉重要性 |
| 全球石油需求 | 约 102.2 mb/d | 2023 | 教学模型分母 |

`mb/d` 表示百万桶/日。不同机构对石油、石油液体、生物燃料和炼厂增益的定义不同，跨来源比较前必须核对口径。

## 弹性实验室

原型使用一阶静态近似：

```text
受影响流量 = 霍尔木兹基线流量 × 通行受阻比例
净供应缺口 = max(0, 受影响流量 − 可替代分流)
缺口率 = 净供应缺口 ÷ 全球石油需求
理论价格变化 = 缺口率 ÷ |短期需求价格弹性|
```

模型只回答“需求越刚性，同样缺口为何需要更大价格变化”。它没有显式模拟库存释放、OPEC+剩余产能、供给响应、需求破坏、制裁执行、金融仓位和预期反馈，因此不能直接用于预测油价。

## 霍尔木兹缺口瀑布

旗舰专题将20.9 mb/d风险敞口拆为五类教学参数：受影响等效流量、管道绕行、应急库存、非海峡增供、需求与炼厂响应。

```text
平衡缺口 = 受影响等效流量
         − 管道绕行
         − 应急库存释放
         − 非海峡增供
         − 需求与炼厂响应
```

管道改变运输路径，库存只是跨期调节，增供改变当期供应，需求响应则通过价格压缩消费。四者经济含义不同，放在同一瀑布中仅用于解释市场如何重新平衡。

## 通胀传导模型

```text
直接 CPI 贡献 = 油价冲击 × 向零售能源传导率 × 能源相关 CPI 权重
总压力 = 直接 CPI 贡献 + 间接成本假设
```

能源权重、传导率和间接成本都由读者显式设置。模型不声称代表中国或其他经济体的官方估计；对中国还需考虑成品油调价、税费、汇率、炼化利润、需求和政策库存。

## 公司比较

公司层面不以油田数量作为核心指标。后续数据表优先采集：

- 证实储量和储量寿命
- 产量、自然递减率与新增项目节奏
- 实现油价、桶油运营成本和完整周期成本
- 资本开支、自由现金流和资产负债率
- 炼化公司的原油加工量、裂解价差和产品结构
- 油服公司的订单、服务价格和客户资本开支

## 原油指纹

原油坐标使用 API 度和质量含硫量两个维度。页面中的 WTI、Brent、Murban、Oman、Arab Light、Mars、Maya 数值为公开资料中的典型代表值，不保证每一批货物完全一致。

教学分类采用简化阈值：

- API ≥ 35：轻质；25—35：中质；API < 25：重质
- 含硫量 < 0.5%：低硫或甜油；其余归入含硫或酸油

这不是完整原油评价。酸值、金属、残炭、馏分曲线、倾点和污染物同样影响炼化价值。

## 炼厂产品倾向模型

模型先按 API 度给出100单位进料的基础产品倾向，再按炼厂转化深度把部分渣油方向性转移到汽油、石脑油、柴油和航煤。

```text
可转化重组分 = min(装置转化参数, 基础渣油倾向 − 最低保留量)
```

输出不是工程收率，也不能用于估算具体炼厂利润。真实结果需要完整原油评价、装置物料平衡、产品规格、氢气和能源成本。

## 到岸成本模型

```text
炼厂到岸成本 = 基准价格 + 品质/地区升贴水 + 运费 + 风险摩擦
```

风险摩擦可包含保险、制裁、信用和融资成本。模型尚未纳入汇率、损耗、港杂费、套期保值和炼厂适配成本。

## 历史冲击指纹

历史时间轴对供给、需求、物流/库存、政策/地缘给出1—5的定性强度，用来帮助比较主导机制。该分数不是计量分解，事件中的价格锚点为名义美元约数。

## 供需平衡与库存投影

供需实验室只计算相对于基线的增量变化，所有输入单位均为 `mb/d`：

```text
每日平衡 = 非OPEC新增供应 + OPEC+行动 − 供应中断 − 全球需求变化
90天库存变化 = 每日平衡 × 90
```

正值表示理论累库，负值表示理论去库。五种预设用于建立数量直觉，不对应 IEA、EIA 或油脉对未来的正式预测。模型没有加入季节性、炼厂维护、数据误差、地区错配和库存可用性。

## 期货期限结构

期限结构实验室采用简化的持有成本关系：

```text
远期价格 ≈ 现货价格 × e^[(融资成本 + 仓储成本 − 便利收益) × 时间]
```

当净持有成本为正时，教学曲线表现为 Contango；便利收益高于融资和仓储成本时，表现为 Backwardation。真实曲线还受库存地点、交割规则、品质、运力、套保需求、季节性和市场预期影响。曲线结构是市场状态证据之一，不是单独的价格方向信号。

## 3-2-1裂解价差

页面先把以美元/加仑报价的成品油乘以每桶42加仑，再计算三桶原油对应的平均产品收入：

```text
3-2-1裂解价差 =
[(2 × 汽油$/gal × 42) + (1 × 馏分油$/gal × 42)] ÷ 3 − 原油$/bbl
```

该数值是标准化毛加工利润代理，不是具体炼厂利润。它没有扣除能源、氢气、催化剂、人工、检修、运输和损耗，也没有表达原油品质、装置配置、产品收率、库存损益和地区税费。

## 中国行业传导矩阵

行业矩阵先区分四种价格情景，再讨论利润方向：需求推动上涨、供应中断上涨、供给过剩下跌、需求崩塌下跌。每个情景同时给出需要验证的库存、裂解、运费、开工或客流信号。

矩阵属于研究框架，不是证券评级。公司层面仍需加入实现价格、套保、产量、成本、税费、政策定价、资产质量、负债和估值位置。页面中的“强受益”“承压”等标签只描述产业位置的方向性暴露。

## 更新节奏

- 咽喉流量、全球供需：每年一次
- 历史价格、库存与产量：每季度或每半年一次
- 供需预设、期限结构与裂解模型：公式变化时复核，至少每年检查一次参数范围
- 行业传导矩阵：产业机制变化或政策定价规则变化时更新
- 公司比较：年报后更新，重大项目变化时补充
- 历史事件与机制文章：事实修订时更新，不追逐日常新闻

每次更新都应记录更新时间、旧值、新值、来源链接和口径变化。
