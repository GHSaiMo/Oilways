# 历史数据与年度快照

## 章节目标

数据长镜头不是行情看板。它用低频年度数据帮助读者建立三个长期直觉：

1. 全球石油市场的关键失衡远小于供应和消费总量。
2. 商业库存与剩余产能都是缓冲，但一个是存量，一个是可增加的流量。
3. 油价变化需要经过资本开支、钻完井和自然递减，才会变成产量。

## 数据版本

- EIA Short-Term Energy Outlook：2026年7月版本
- 年度范围：2010—2025
- 美国原油历史表发布日期：2026-06-30
- 公司快照：2024年报

网站不调用实时接口。源数据整理后写入 `src/data/historicalData.ts`，更新时必须整段替换并记录版本，避免不同月份的STEO估计混在一张图中。

## 图表边界

### 全球供应与消费

两者之差称为“年度表观差”，不称为确定累库或去库。库存统计、海上流量、炼厂增益、统计误差和地区错配都会造成偏差。

### OECD库存与OPEC剩余产能

库存是年末存量，剩余产能是可增加的日产流量，不能直接加总或比较绝对大小。高库存也不代表油种、地点和释放权限都匹配；高剩余产能也不代表可以当天启动并运达目标炼厂。

### WTI与美国原油产量

图表使用双轴，只展示方向和时间关系，不用线条交叉位置表达统计相关性。美国总产量包含页岩、常规陆上和海上生产。

## 公司快照

第一版覆盖中国海油、中国石化、ExxonMobil和SLB，分别代表海上上游、炼化销售偏重、全球综合油企和油田服务。

快照不建立统一分数。产量、桶油成本、炼厂加工量、EBITDA和自由现金流属于不同商业问题，只有在同一家公司连续年度比较时才具有较强可比性。

公司数据来源入口：

- [CNOOC Limited / Investor Relations](https://www.cnoocltd.com/english/investorrelations/)
- [Sinopec Corp. / Annual Report 2024](http://www.sinopec.com/listco/en/000/000/066/66242.shtml)

## 更新步骤

1. 下载新的STEO工作簿，并记录报告月份。
2. 核对序列名称、单位和OPEC成员定义是否变化。
3. 用同一版本更新2010年以来的全部年度值。
4. 检查表观差、极值年份和单位换算。
5. 年报发布后逐家公司更新，不把不同年份混在同一快照中。
6. 完成桌面与390px视觉验收，确认坐标、来源和注释仍可阅读。
