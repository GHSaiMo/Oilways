# 油脉产品路线

## 北极星

油脉不是油价资讯站，也不是石油名词百科。它要帮助读者形成一条稳定的分析顺序：

> 物理流量 → 供需缺口 → 价格形成 → 通胀传导 → 行业利润 → 资产价格

最重要的认知结果有两个：

1. 石油是现代交通、材料、农业和工业系统的重要底层投入。
2. 霍尔木兹海峡是流量高度集中、替代能力有限的全球能源咽喉。

## 目标读者

### 第一级：完全陌生

通过真实影像、日常物品和一桶油的旅程，回答“石油为什么与我有关”。

### 第二级：普通投资者

通过因果链、历史案例和交互模型，回答“新闻如何传导到通胀和行业利润”。

### 第三级：专业投资者补全

通过地缘地图、口径注释、成本定义和期限结构，补足常被金融分析简化的物理世界。

## 内容架构

| 模块 | 核心问题 | 主要表达 |
| --- | --- | --- |
| 石油就在身边 | 石油为什么不可忽视 | 物品矩阵、生活路径 |
| 一桶油的诞生 | 不同原油为何不同 | 剖面、品级、成本定义 |
| 全球流动系统 | 局部中断为何全球定价 | 地图、航线、咽喉 |
| 炼厂如何工作 | 原油如何成为产品 | 流程图、产品收率 |
| 油价如何形成 | 新闻中的“油价”是什么 | 基准油、升贴水、曲线 |
| 供需为什么失衡 | 小缺口为何大波动 | 库存、弹性、剩余产能 |
| 石油重塑历史 | 地缘冲突如何改变秩序 | 时间轴、事件解剖 |
| 从油价到资产 | 如何形成投资判断 | 传导矩阵、公司类型 |

## 页面模板

每个专题统一提供三层信息：

1. **一分钟结论**：一个核心判断、三个关键数字、一条因果链。
2. **十分钟机制**：图解、案例、对照和常见误区。
3. **专业深读**：定义、公式、历史数据、来源与口径差异。

每个动态事件统一采用五问模板：发生了什么、影响多少桶、持续多久、谁能填补、如何传导。

## 两周原型计划

### 第 1—2 天：品牌与认知入口

- 确立油脉品牌、首页主张与视觉系统
- 完成霍尔木兹首屏及核心数字口径
- 确立“叙事 / 深读”双层阅读机制

### 第 3—5 天：物理世界

- 完成一桶油六站旅程
- 完成生活用途与产业链总览
- 完成五个能源咽喉交互地图

### 第 6—8 天：价格与推演

- 完成供需、库存和价格弹性解释
- 完成霍尔木兹中断压力测试
- 加入参数、公式、限制条件与来源

### 第 9—10 天：投资者桥梁

- 完成物理冲击到资产价格的四步框架
- 完成上游、炼化、油服与下游行业矩阵
- 加入 A/H/美股代表性公司类型比较

### 第 11—12 天：内容治理

- 建立来源登记、数据年份和更新时间规则
- 校对所有数字及术语
- 增加模型与投资教育免责声明

### 第 13—14 天：体验验收

- 桌面与移动端视觉检查
- 键盘操作、对比度和减少动画模式检查
- 构建验证、性能优化与发布准备

## 原型之后

原型按“旗舰地缘案例 → 原油与炼厂 → 市场结构 → 历史数据 → 投资应用”的顺序推进。当前已完成前五个内容阶段；后续继续加强事件档案和可维护的数据治理，不引入高频行情。

## 第二阶段完成情况

霍尔木兹旗舰专题已经完成第一版，包含：

- 风险敞口到平衡缺口的四级情景瀑布
- 市场、物流、库存、宏观四只危机时钟
- East–West、Habshan–Fujairah 等替代管道限制
- 亚洲暴露与库存可用性的四道门
- 油价到 headline CPI 的透明参数模型
- 用六项硬信号区分标题、物流、实货和系统冲击的事件验证器

下一阶段将从“霍尔木兹单点案例”扩展到石油体系本身，优先建设原油品质与炼厂、全球定价基准、历史石油危机三部分。

## 第三阶段完成情况

原油、炼厂、定价和历史周期已经形成第一版系统内容：

- 用 API 度和含硫量构建七种代表原油的互动坐标
- 用分离、转化、处理三步解释炼厂的基本功能
- 用三类炼厂配置展示重油折价如何被装置能力转化
- 区分 WTI、Brent、Dubai/Oman 和 Murban 的地点、品质与交割机制
- 用透明公式拆解炼厂到岸原油成本
- 用统一冲击指纹比较七次石油危机

下一阶段优先补充供需与库存历史数据、期货期限结构、裂解价差，以及中国油气产业链的行业利润传导。

## 第四阶段完成情况

供需、库存、期限结构、裂解价差和中国行业利润传导已经形成第一版：

- 用五种供需预设和四个透明参数解释每日微小失衡如何累积为库存变化
- 用90天投影建立 `mb/d` 与百万桶库存之间的数量直觉
- 用融资、仓储和便利收益构造12个月教学曲线
- 区分 Contango、近似平坦和 Backwardation，但不把曲线单独写成涨跌预测
- 用统一单位计算3-2-1裂解价差，并明确它只是毛加工空间代理
- 用四种市场情景比较上游、炼油、化工、油服、油轮和航空物流的利润方向
- 将旧研究库中的“月差、库存、炼厂约束和A股映射”沉淀为长期框架

下一阶段优先建设低频历史数据层：全球供需与库存长图、OPEC+闲置产能、美国页岩油响应、A/H/美股代表公司年度快照，以及每项数据的更新时间和口径记录。

## 第五阶段完成情况

低频历史数据层和第一版公司年度快照已经完成：

- 固定EIA 2026年7月STEO版本，沉淀2010—2025全球供应与消费年度序列
- 用同一时间轴比较OECD商业库存和OPEC剩余原油产能，区分存量与流量缓冲
- 对照WTI年度均价与美国原油产量，解释价格、资本开支和产量之间的滞后
- 为中国海油、中国石化、ExxonMobil和SLB建立2024年报快照
- 明确公司之间的产量、成本、利润和现金流口径不可直接排名
- 将全站最小字号提升至10px，正文型辅助信息提升至11—12px

下一阶段优先建设“事件档案与复盘层”：把当期地缘事件按流量、持续时间、库存、价差、政策和资产反应归档，并允许回看事件假设后来是否被数据证实。

## 成功指标

- 读者能够解释霍尔木兹重要性的物理原因，而不是只记住一个百分比。
- 读者面对油价新闻时，会先询问流量、持续时间、库存和替代能力。
- 读者能够区分上游、炼化、油服公司的油价敏感度。
- 内容被引用时，关键数字能回到明确年份和原始来源。
